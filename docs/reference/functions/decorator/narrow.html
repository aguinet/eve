**Expressive Vector Engine** `eve::pedantic`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the narrow decorator. Member functions ==================================================================================================== Performs the operation defined by the callable [element-wise](../../../glossary.html#element-wise_) on narrowed arguments. Parameters ---------------------------------------------------------------------------------------------------- `x` : an arithmetic callable Return value ---------------------------------------------------------------------------------------------------- The callable is supposed to be an arithmetic call. narrow makes the callable automatically try to convert with saturation all its remaining parameters to a type whose elements are convertible to the first type element before call, but insist on this being a narrowing operation. It does not compile if the first type has an element type strictly promoting the common compatible type of the other parameters.
